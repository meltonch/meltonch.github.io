UpNext: A Centralized Planning Application

About

UpNext is a web app built around the idea of centralizing all the different utilities one might use to keep their life organized. Included utilities include a calendar with weekly/monthly views, customizable lists, and a weather widget that updates in real time. I developed this application as part of a team for a term project in Web Programming.

Graphics: Line Projection

About

This was a project demonstrating two different algorithms for line projections. It's written in Java, and the interface is designed and implemented in JavaFX. I built the project using Scala SBT with nothing more than a simple "sbt compile/sbt run" as needed from the home directory on a command line. Once running, simply input the number of lines you want to draw, select an algorithm, and click "draw."

Graphics: 3D Projection

About

This is an expansion of the line drawer project. You'll notice it's named "Mag 6 3D" because I thought it was cool but it really isn't so I apologize for that. It's also built and run using Scala SBT. Once running, you must enter eye coordinates (the "camera" which your image is being viewed from), the ideal viewing distance from the screen, and the screen size. Otherwise, you'll end up with null pointer exceptions, and the program won't run correctly. This project is also written in Java with an interface designed and implemented in JavaFX. You can load pre-designed shapes by clicking the "load file" button. Transformations can also be applied, but all fields for the transformation must be filled in for it to actually be applied.
