Required Files:
	game1.html
	game1.js
	dude.js
	alien.js
	laser.js
	img/ship1.png
	img/laser.png
	img/alien.png

Running:
	The game can be run by placing all files in the same folder, with the image files (.png) being placed
	in a subfolder named "img". The game can then be launched by opening game1.html from a web browser.
